## ting-hexo

`ting-hexo` is a ready-to-run Hexo content demo site for showcasing common blog capabilities, including posts, tags, categories, search, table of contents, code blocks, math formulas, and encrypted posts.

### Node.js

> [NVM](https://github.com/nvm-sh/nvm)

```sh
nvm install && nvm use
```

### Scripts

```sh
npm run server
npm run build
npm run clean
```
