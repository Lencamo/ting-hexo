## 代码块示例

下面这个片段模拟站点初始化阶段的配置合并逻辑，适合顺手测试代码高亮和复制按钮：

```ts
type FeatureFlags = {
  search: boolean
  mathjax: boolean
  encryptedPost: boolean
}

function createSiteFlags(overrides: Partial<FeatureFlags> = {}): FeatureFlags {
  return {
    search: true,
    mathjax: true,
    encryptedPost: true,
    ...overrides
  }
}

console.log(createSiteFlags({ encryptedPost: false }))
```

如果你更习惯命令行，也可以直接用一段 shell 文本验证另一种代码块样式：

```sh
hexo clean
hexo generate
hexo server
```

## 数学公式示例

库存周转天数可以简化表达为 `库存 / 日均消耗`。如果用公式表示：

$$
turnoverDays = \frac{currentStock}{dailyUsage}
$$

当 `turnoverDays < 7` 时，页面上通常会追加预警提示；当 `turnoverDays >= 14` 时，可以把补货优先级降为普通。
